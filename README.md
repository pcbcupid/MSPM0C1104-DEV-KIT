# MSPM0C1104-DEV-KIT / INDUSTRIAL TEMPERATURE/HUMIDITY SENSOR MODULE WITH RS485 OUTPUT 
![pcbcupid-gsense-gsi003](https://github.com/pcbcupid/MSPM0C1104-DEV-KIT/blob/main/Images/pcbcupid-gsi0003.avif)
